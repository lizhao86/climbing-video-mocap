# IMG_6424 攀岩动作分析

单机位攀岩视频姿态分析的完整产物。基于 MediaPipe 姿态估计（相机静止，单目）。

视频：52.9 秒 · 2655 帧 @ 50fps · 人体检出率 100%。

## 怎么看

**先打开 [`攀岩动作报告卡.html`](攀岩动作报告卡.html)** —— 双击即可，含核心指标、图表、crux 截图和解读建议。其余文件是它的原始数据和素材。

## 关键结论

- 净上升约 6.7 身长，攀爬效率 20.5%（净上升÷重心总走线），横向移动多于纵向。
- 台阶式上升，节奏型攀爬；35% 时间在静止找点/锁定。
- 全程 18 个 move；**右脚移动最少（仅 3 次），蹬脚偏左**。
- 流畅度评分 44.9/100（中等），重心多次急起急停。
- 自动识别 5 个吃力点（crux）：4.9 / 7.8 / 19.4 / 33.8 / 49 秒。

## 数据口径

单摄像头 + 相机静止假设。长度/速度以「身长」为单位（尺度无关），反映**相对趋势与节奏**，非实验室级绝对测量；单目对深度方向不敏感，分析以画面内运动为主。需要真实米制 / 深度 / 可导 Blender 的三维数据，建议多机位或 GVHMR 等世界坐标方案。

## 复跑

```bash
# 1) 提取骨架（环境变量控制缩放/抽帧/精度）
CLIMB_MAX_W=960 CLIMB_COMPLEXITY=2 python3 脚本/climb_pose.py <视频路径>
# 2) 算指标 + 出图（脚本内路径需指向上面生成的 CSV）
python3 脚本/analyze_climb.py
```
依赖：`mediapipe opencv-python numpy matplotlib`，以及系统 `ffmpeg`（用于读取旋转标记）。
